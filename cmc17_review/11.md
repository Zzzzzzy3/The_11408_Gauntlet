==积分区域对称与交换积分顺序==

对于

$$\int_a^bf(x)(\int_a^x g(s)ds)dx$$

有

$$\int_a
^b g(s)(\int_a^x f(x)dx)ds$$

---


$$\begin{aligned}
& \text{六、(本题 14 分) 设函数 } f(x), g(x) \text{ 在 } [a, b] \text{ 上连续可导且满足以下三个条件：} \\
& \text{(i) } \int_a^b f(x)  dx = \int_a^b g(x)  dx = 0; \\
& \text{(ii) } f'(x)g(x) - f(x)g'(x) > 0, \quad x \in [a, b]; \\
& \text{(iii) } g(x) \text{ 在 } (a, b) \text{ 上有唯一零点 } \xi。 \\
& \text{证明：(1) } \int_a^b f(x) \left( \int_a^x g(s)  ds \right)  dx = -\int_a^b g(x) \left( \int_a^x f(s)  ds \right)  dx。 \\
& \qquad \ \text{(2) } \int_a^b f(x) \left( \int_a^x g(s)  ds \right)  dx > 0。
\end{aligned}$$ 

解(1)

$$\begin{aligned} 
&\text {由分部积分}
\\
&\quad\int_a^bf(x)(\int_a^xg(s)ds)dx=\int_a^bf(x)G(x)dx=\int_a^bG(x)d[F(x)]dx
\\
&\text{即}
\\
&\quad F(x)G(x)|^b_a-\int_a^bg(x)F(x)
\\
&\text {由(i)可知}\quad F(x)G(x)|^b_a=0 
\\
\text{即证}
\\
\end{aligned}
$$



解(2)

$$\begin{aligned}
&\text{由积分区域易得：} \\
&\quad \int_a^b f(x)\left( \int_a^x g(s)  ds \right) dx 
    = \frac{1}{2} \iint_{D_0} [f(x)g(s) - f(s)g(x)]  dx  ds \\
&\quad \text{其中区域 } D_0 = \{(x,s) \mid a \leq x \leq b, a \leq s \leq x\} \\
\\
&\text{由(ii)可得：} \\
&\quad m(x) = \frac{f(x)}{g(x)}, \quad m'(x) \geq 0 \quad (\text{在 } g(x) \neq 0 \text{ 处}) \\
\\
&\text{由(iii)可得：} \\
&\quad \text{不妨设 } g(x) \text{ 在 } x \in [a,\xi] \text{ 上 } g(x) \leq 0, \quad x \in [\xi,b] \text{ 上 } g(x) \geq 0 \\
\\
&\text{将 } D_0 \text{ 分成 } D_1, D_2, D_3： \\
&\quad D_1 = \{(x,s) \mid a \leq s \leq x \leq \xi \leq b\} \\
&\quad D_2 = \{(x,s) \mid a \leq s \leq \xi \leq x \leq b\} \\
&\quad D_3 = \{(x,s) \mid a \leq \xi \leq s \leq x \leq b\} \\
\\
&\text{由单调性易得：} \\
&\quad \text{在 } D_1, D_3 \text{ 上，} f(x)g(s) - f(s)g(x) \geq 0 \\
\\
&\text{在 } D_2 \text{ 区域分段积分和(i)可得：} \\
&\quad \int_a^b f(x)\left( \int_a^x g(s)  ds \right) dx 
    = \int_a^b f(x)\left( \int_a^{\xi} g(s)  ds \right) dx \\
&\quad \text{即} \\
&\quad \frac{1}{2} \iint_{D_2} [f(x)g(s) - f(s)g(x)]  dx  ds = 0 \\
\\
&\text{即证}
\end{aligned}$$
